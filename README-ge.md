# ანტიმიკრობული პეპტიდების პროგნოზირების მოდელების გაუმჯობესება და შედარებითი ანალიზი

## პროექტის აღწერა
ანტიმიკრობული პეპტიდები (*Antimicrobial Peptides – AMP*) წარმოადგენს მოკლე ამინომჟავურ
მიმდევრობებს, რომლებიც მნიშვნელოვან როლს ასრულებენ მიკროორგანიზმების
წინააღმდეგ ბუნებრივ იმუნურ სისტემაში. ბოლო წლებში ისინი აქტიურად განიხილება,
როგორც ანტიბიოტიკების პოტენციური ალტერნატივა, განსაკუთრებით ანტიბიოტიკების
მიმართ რეზისტენტული ბაქტერიების გავრცელების ფონზე.

ამ მიმართულებით კვლევებისთვის მნიშვნელოვანი რესურსია ***DBAASP***, რომელიც შეიცავს
ანტიმიკრობული პეპტიდების ამინომჟავურ მიმდევრობებსა და მათ ექსპერიმენტულად
დადგენილ ბიოლოგიურ აქტივობას. აღნიშნული მონაცემები ფართოდ გამოიყენება
სხვადასხვა მანქანური სწავლების მოდელის შესაქმნელად, რომლებიც პროგნოზირებენ
პეპტიდის ანტიმიკრობულ აქტივობას მისი მიმდევრობის საფუძველზე.

პროექტის მიზანია ანტიმიკრობული აქტივობის პროგნოზირების არსებული მოდელების
გაუმჯობესების შესაძლო გზების შესწავლა. ამ მიზნით გამოყენებული იქნება ***DBAASP***
მონაცემთა ბაზის მონაცემები, რომლებზეც შეიქმნება და გაიწვრთნება სხვადასხვა
პროგნოზირების მოდელი. ასევე დაიტესტება განსხვავებული მიდგომები, მათ შორის
თანამედროვე მეთოდები, როგორიცაა ცილოვანი მიმდევრობების წარმოდგენა წინასწარ
გაწვრთნილი მოდელების საშუალებით (*Protein Language Models*).

### პროექტის ფარგლებში განხორციელდება შემდეგი ეტაპები:
- ***DBAASP*** მონაცემთა ბაზიდან შესაბამისი მონაცემების შერჩევა და დამუშავება;
- ანტიმიკრობული აქტივობის პროგნოზირების მოდელების შექმნა და გაწვრთნა;
- სხვადასხვა მიდგომის ექსპერიმენტული შეფასება;
- მიღებული მოდელების შედარება არსებულ პროგნოზირების მეთოდებთან;
- ანალიზი, თუ რომელმა მეთოდებმა მოახერხა პროგნოზირების გაუმჯობესება და
რომელმა არა.

პროექტის შედეგი იქნება პროგნოზირების მოდელების შედარებითი ანალიზი. აღნიშნული
კვლევა ხელს შეუწყობს არსებული მიდგომების უკეთ გაგებას და შესაძლო გაუმჯობესების
მიმართულებების გამოვლენას.
